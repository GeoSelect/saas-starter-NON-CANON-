## Notes

- **Footer links are plan-aware**: Anonymous, Basic, Pro, and Enterprise users see different navigation menus
- **Dynamic routes** include parcel listing pages and workspace navigation
- **Demo mode** buttons include query parameters: `?demo=authenticated`
- **Direction buttons** in Search View use query parameters: `?view=north|east|south|west`
- All navigation uses Next.js `<Link>` component for client-side routing except where explicitly noted with `window.location.href` or `<a>` tags
